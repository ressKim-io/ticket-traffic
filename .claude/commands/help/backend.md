# Backend Commands

Java/Kotlin 백엔드 개발을 위한 명령어입니다.

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/backend review` | 코드 리뷰 |
| `/backend test-gen` | 테스트 코드 생성 |
| `/backend api-doc` | OpenAPI 문서 생성 |
| `/backend refactor` | 리팩토링 제안 |

## 관련 Skills

| Skill | 내용 |
|-------|------|
| `/spring-data` | JPA, QueryDSL 패턴 |
| `/spring-cache` | Redis 캐싱 전략 |
| `/spring-security` | Security, OAuth2, JWT |
| `/spring-testing` | JUnit, Mockito, Testcontainers |
| `/concurrency-spring` | 동시성 문제 해결 |
| `/api-design` | REST API 설계 |

## Quick Reference

```bash
# Gradle
./gradlew test
./gradlew build

# Maven
mvn test
mvn package
```
